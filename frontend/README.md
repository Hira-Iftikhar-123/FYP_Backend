#  Frontend (React Native + Expo)

##  Overview
This folder contains the **frontend** of the project, built using **React Native**, **Expo**, **Tailwind (via NativeWind)** and **TypeScript**.  
The app provides the mobile UI for viewing camera feeds, reporting incidents, and interacting with backend services.

---

## Prerequisites
Make sure the following is installed:

- A smartphone connected to the **same Wi-Fi network** as your PC and with the expo go app installed

---

##  Installation

Because the project includes packages with peer dependency conflicts, you must install using the legacy peer dependency flag:

```sh
npm install --legacy-peer-deps
